
# PSD and Figma Guide

## What is PSD?
PSD (Photoshop Document) is Adobe's native file format for creating and editing digital designs, images, and layouts. It supports layers, effects, and advanced editing tools.

## What is Figma?
Figma is a cloud-based UI/UX design tool that enables collaborative design work in real-time. It's browser-based and supports prototyping, wireframing, and design systems.

## PSD to HTML Conversion

1. **Open PSD in Photoshop** → Export layers as PNG/JPG
2. **Use online converters** (PSD2HTML, CloudConvert)
3. **Manual coding** → Slice design, write HTML/CSS
4. **Use Photoshop plugins** → Auto-generate HTML
5. **CSS frameworks** → Bootstrap, Tailwind for responsive design

## Figma to HTML Conversion

1. **Figma plugins** → HTML exporter, Anima, Relume
2. **Design tokens** → Export components as code
3. **CSS generation** → Use Figma dev mode
4. **Component libraries** → React, Vue exports
5. **Manual mapping** → Recreate design in code
