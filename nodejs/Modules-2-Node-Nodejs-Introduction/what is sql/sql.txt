4) truncate : truncate is used to empty all data or delete all data from tables

            truncate table flipkart.flip_categories;


5) drop : drop is used to delete database and its structures also 
          note : after drop we can not rollback structures 

          drop is also used to delete tables structures after drop we can not rollback table structure 

          example : drop database  flipkart; 
          example: drop table flipkart.flip_categories;
