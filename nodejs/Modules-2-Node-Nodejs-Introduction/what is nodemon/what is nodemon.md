what is nodemon ?
 nodemon is an library of node js 
 nodemon is install via npm install nodemon 
 nodemon is automatically restart the server
 nodemon registered inside of package.json 

   "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev":"nodemon index.js"
  },


 run node js  via nodemon

 cmd : npm run dev
